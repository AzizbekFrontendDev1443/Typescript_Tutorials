Typescript is a superset of Javascript that adds optional type annotations and other features such as interfaces, classes and namespaces. Javascript is dinamically-typed language that is primarilly used for client-side web develoment and can also be used for server-side development.
Types: Typescript has optional type annotations while Javascript is dinamically-typed. This means that in Typescript, you can specify the data-type of variables, parametrs and return values, which can help catch type-related errors at compile-time
Syntax: Typescript extends Javascript syntax with featurs like interfaces, classes and namespaces. This provides a more robust and organized structure for large-scale projects.
Tooling Typescript has better tooling support, such as better editor integration, type cheking, and code refactoring.
Backwards Compatibility: Typescript is fully compatible with exist Javascript code, which means you can use Typescript in any Javascript environment.
